Installing and setting up bspwm

bspwm is simple , it just needs 2 config files : 1) bspwmrc 2) sxhkdrc . Install the packages for your distro :

sudo pacman -S feh bspwm sxhkd maim xclip picom ttf-font-awesome

clone my repo for the configs :

git clone https://github.com/siduck76/bspwm-dotfiles


Oki so to run bspwm you need two files : bspwmrc and sxhkd . bspwmrc file is needed for auto-starting apps , like if you want browser to open up straight after you start up system then you have to put the browser's run command in the bspwmrc file , like for firefox you would have to add firefox & at the end of the bspwmrc file !.

Now the second file which is sxhkdrc is needed for keybinds ( keyboard shortcuts ) like if you want if to open terminal by mod ( windows key ) + enter then you would have to specify it in the sxhkdrc file. example :



# Now copy the bspwmrc / sxhkdrc files ( from my repo which you cloned )

mkdir ~/.config/bspwm && mkdir ~/.config/sxhkd

# copy the bspwmrc file to bspwm folder and sxhkdrc file to the sxhkd folder

chmod +x bspwmrc ( this is important since it runs whole system )


# apply wallpaper ( feh should be installed )

# put this line below in your bspwmrc

feh --bg-fill yourwallpaper
